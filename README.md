# ElevEd – Smart AI Learning Platform

ElevEd is a smart AI-powered learning platform designed to personalize the educational experience for students. The system adapts learning content based on individual learning styles and predicts the time required for a student to master a topic. The goal of ElevEd is to make learning more effective, engaging, and personalized.

---

## 📌 Project Overview

Traditional education systems follow a one-size-fits-all approach where the same content is delivered to every student in the same way. However, every student has a different learning style and learning speed.

ElevEd solves this problem by using Artificial Intelligence to analyze how students learn and adapt the learning experience accordingly. The platform identifies the learner’s style, predicts their progress, and provides personalized feedback to improve learning outcomes.

---

## 🚀 Key Features

### 1. Learning Style Detection AI

The system analyzes how students interact with learning materials and determines their learning style.

Based on the detected learning style, the platform provides suitable learning content such as:

* Visual explanations (charts, diagrams)
* Practical exercises
* Concept-based explanations

This makes learning more effective and enjoyable.

---

### 2. Time-to-Mastery Predictor

This feature acts like a personal learning coach. It evaluates:

* Student performance
* Accuracy of answers
* Time taken to solve problems

Using this data, the system predicts how long a student may take to master a topic.

Example prediction:

> "You can complete Data Structures in 7 days if you study 1 hour daily."

This helps students track their progress and stay motivated.

---

### 3. Peer Twin AI (Future Scope)

Peer Twin AI is a future concept where the platform creates a virtual version of the learner that reflects their knowledge level.

Students can interact with this twin by:

* Solving questions
* Practicing problem-solving
* Debating answers

This interactive method helps learners identify weak areas and improve through practice.

## ⚙️ Installation and Setup

### 1. Clone the Repository

```
git clone https://github.com/Sibangi-2911/ElevEd---Personalised-Al-Learning-Platform.git
```

### 2. Navigate to the Project Folder

```
cd ElevEd---Personalised-Al-Learning-Platform
```

### 3. Install Dependencies

```
npm install
```

### 4. Start the Server

```
node server.js
```

### 5. Run the Frontend

Open the `index.html` file in your browser.
